Positionally encoded U-Net for robust cortical parcellation (v1.0.0)
usage: python predict.py <path_images> <path_predictions>

path_images is a directory containing FS subjects, each of which has a
surf directory with parameterizations inflated.H.mgz, surf.mgz,
curv.mgz

path_predictions is a directory to which the parcellation predictions
are stored.
